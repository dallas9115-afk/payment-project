🔵 모드 A (PortOne SDK)
클라이언트에서 직접 PortOne 결제창 열기
위 주문 목록에서 "결제하기" 버튼을 클릭하면 자동으로 진행됩니다:
  1. 서버에 결제 시작 요청 (PENDING 상태로 DB 저장)
  2. PortOne 결제창 열기
  3. 사용자 결제 진행
🟢 모드 B (API 수동 테스트)
백엔드 API 엔드포인트 직접 호출
결제 확정/취소 API를 직접 테스트하려면 결제 ID를 입력하세요.
📋 사용 안내

일반 사용자 흐름:

  1. 상점에서 주문 생성
  2. 위 주문 목록에서 "결제하기" 버튼 클릭
  3. PortOne 결제창에서 결제 진행

개발자 테스트 (모드 B):

  1. 결제 완료 후 받은 결제 ID 입력
  2. "결제 확정" 또는 "결제 취소/환불" 버튼으로 API 테스트
